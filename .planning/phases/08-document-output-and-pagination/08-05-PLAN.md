**Test 1: PDF Export Optimization**

4. Check file size - should be under 5MB for text-only content
5. Open the PDF - text should be readable, no major visual artifacts

**Test 2: Word Export**
1. Create a document with:
   - H1 heading
   - Paragraph with bold and italic text
   - Bullet list
   - Numbered list
   - Table (3x3)
   - Callout block (if available)
2. Click Export > Word (.docx)
3. Save the file
4. Open in Microsoft Word, Google Docs, or LibreOffice
Expected: All elements preserved with proper formatting

**Test 3: Pagination Mode**
1. In toolbar, find the pagination toggle
2. Click to enable pagination (Pages: On)
3. Document canvas should show visual page breaks (gray lines)
4. Change page size to Letter - canvas width and page spacing should adjust
5. Change to Legal - same behavior
6. Print preview (Cmd+P) - pages should break cleanly, not cutting headings
7. Disable pagination (Pages: Off) - page boundaries should disappear
Expected: Visual page boundaries match selected page size

**Test 4: Markdown Source View**
1. Create a document with heading, paragraph, bold text, list
2. Press Cmd+/ (or Ctrl+/ on non-Mac)
3. Should switch to Markdown source view with syntax highlighting
4. Verify Markdown shows: # Heading, **bold**, - list items
5. Make an edit in source (e.g., add a word)
6. Press Cmd+/ again to switch back
7. Edited content should appear in rendered view
Expected: Bidirectional sync between rendered and source views
